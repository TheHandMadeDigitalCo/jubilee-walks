About

Digitised versions of the Corfe Mullen Diamond Jubilee walks created by Councillor Wendy Dix, which are originally located on the Corfe Mullen Town Council site.

The maps were produced using QGIS software to trace the routes over Open Street Map data.

Data Availability

The data is consumable in a few ways (and downloadable from the links below):

  • GPX files: these should be useful on most devices/GIS software
  • GeoJSON: a set of JSON documents that are mainly useful for web but also for GIS software
  • PDF: a backup of the original PDF files
  • Mapping Application: a simple web map application that anyone can use

In addition to the above files/applications all source code (and data) is available on GitHub in the Hand Made Digital Co. repository.

Application

A mapping application is available to preview and use for walks.
